Задание:
"Распознавание печатных цифр
Создать датасет для распознавания печатных цифр от 0 до 9 (изображения 8бит, 28x28 пикселей, предполагается, что шрифт цифр один).
Обучить нейронную сеть на данном датасете – классификатор для распознавания цифр.

Инструменты для создания датасетов – любые.
Языки программирования С++, С#, Python (на выбор).
Фреймворки для нейросетей Caffe или Tensorflow (на выбор).
Нейронную сеть можно взять готовую или разработать свою."

Решение:
1. Написал в Ворде последовательности цифр: обычную, жирную, курсив (ее позже решил не брать)
2. Нарезал последовательности на скриншоты и загрузил в папку original_images
3. Создал файл augmentation.py, который производит аугментацию подготовленных изображений и новых изображений по папкам
4. Добавил модель model/lenet.py
5. Создал файл lenet_model.py, который ссылается на model/lenet.py как на библиотеку. На выходе я получил график
Потерь/Точности и обученную модель saved_model/lenet_model.hdf5
6. Создал файл lenet_model.py, который загружает обученную модель saved_model/lenet_model.hdf5 и использует ее для
демонстрации работы сети. (Появившиеся окна закрываются на клавишу "Q")
